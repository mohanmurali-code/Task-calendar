# Task-calendar

A small static calendar app built with HTML, JavaScript, and Tailwind loaded from the CDN.

## Run locally

Open `index.html` directly in a browser, or serve the folder locally:

```powershell
node server.js
```

Then visit:

```text
http://localhost:8000
```

## Files

- `index.html` redirects to the calendar page.
- `taskcalendar.html` contains the app UI, styling, and JavaScript.
